# BingoChain Stellar, frontend

React 18 + Vite + TypeScript dapp for the `bingo` Soroban contract on Stellar
testnet. For what the game is, the architecture, the workshop level checklist,
and the demo script, see the [root README](../README.md).

## Setup

```bash
pnpm install
cp .env.example .env
```

`.env` is git-ignored. `.env.example` already points at the live testnet
contract; no secret keys belong in either file, since anything in a `VITE_`
variable is bundled into the public build.

## Dev server

```bash
pnpm dev
# open http://localhost:5173
```

## Test

```bash
pnpm test        # vitest run, frontend/src/lib/commit.test.ts and errors.test.ts
```

## Typecheck and build

```bash
pnpm typecheck
pnpm build        # outputs to dist/
pnpm preview      # serve the production build
```

## Layout

- `src/lib/wallet.ts`, `horizon.ts`, `tx.ts`: Freighter connect, XLM balance,
  classic payments (Level 1).
- `src/lib/contract.ts`, `commit.ts`, `errors.ts`, `events.ts`: the generated
  `bingo-client` bindings, sealed board commitments, the error taxonomy, and
  contract event streaming (Level 2 and 3).
- `src/hooks/*.ts`: one hook per concern (`useWallet`, `useBalance`, `useTx`,
  `useArenas`, `useArena`).
- `src/components/*.tsx`: `Header`, `SendXlmCard`, `Lobby`, `CreateArenaForm`,
  `BoardSetup`, `GameRoom`, `RevealPanel`, `EarningsCard`, `TxStatus`.
- `src/styles.css`: the whole design system, "the back room" felt-table look.
- `packages/bingo-client`: TypeScript bindings generated by
  `stellar contract bindings typescript`, regenerated by `../scripts/deploy.sh`.
